# xcore.swift
Cocoa Touch Classes

## Installation
You have two options to include the `xcore.swift` in your project:

### [CocoaPods](http://cocoapods.org/)

#### Podfile

```ruby
pod 'Xcore', '~> 1.0'
```

### Manual Installation
Drag the classes from the *Sources* folder directly into your project.
